## Setting up SMS

<Info>Setting up sms, requires external api keys for sms sending services</Info>

After navigating to sms set up, from settings you will see a page like this:

![Example Flow](/images/guides/product-manual/t.png)

We currently support sms sending through twillio

## Creating SMS Messages

To create an SMS that you can use in subsequent journey, you need to first navigate to message templates 

![Example Flow](/images/guides/product-manual/mt.png)

Now you need to name the template

![Example Flow](/images/guides/product-manual/nst.png)

Here you can create the message you want to send, along with personalize fields like name

![Example Flow](/images/guides/product-manual/st.png)

## Using sms in Journeys

Once sms is set up and you have created sms templates, you can use them in a journey by dragging an sms step onto the canvas and then selecting the template you designed.

![Example Flow](/images/guides/product-manual/sinj.png)
